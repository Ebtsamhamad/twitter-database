# Twitter Database Project

## Description
Database design simulating core features of Twitter (X) using MySQL.

## Tables
- users
- profiles (One to One)
- tweet (One to Many)
- follow (Many to Many - self relationship)
- like_user (Many to Many)

## Features
- Users can post tweets
- Users can follow other users
- Users can like tweets

## Tools
- MySQL
- MySQL Workbench
